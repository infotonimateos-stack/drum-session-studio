# Caso Orbidi — Documentación legal

## Contexto
- **Contrato:** Acuerdo de Colaboración firmado 16/01/2025 entre G97 Tech Marketing SL (Orbidi) y Groove Factory Studios SL
- **Objeto:** Referencia de clientes para Kit Digital (Puesto de Trabajo Seguro y otros servicios)
- **Situación:** Orbidi comunica resolución unilateral el 10/03/2026 con preaviso de 30 días, restringiendo indebidamente la actividad durante el preaviso
- **Clientes referidos:** +1.000
- **Jurisdicción:** Juzgados de Barcelona (Cláusula 10.1)
